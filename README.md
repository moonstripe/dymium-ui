# Example Interface

For Igor @ Dymium.

### Usage

Install and run the dev command that corresponds to your JS runtime.
It will:

1. Start a dummy server that spews events
2. Start the Live Log Table example interface with Vite.

Node/NPM:

```sh
npm i
npm run dev:node
```

Bun:

```sh
bun i
bun run dev:bun
```
